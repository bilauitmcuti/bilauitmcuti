## Reporting vulnerabilities

- Open a public GitHub issue for security bugs.
- Submit a report via the feedback form: [/feedback](/feedback).
- Contact via email: hello@bilauitmcuti.com.
- Expect a response within 72 hours.

## Privacy & no sensitive data

- This app does not have user accounts.
- No personal information is intended to be stored or transmitted.
- We use Cloudflare Analytics and Google Analytics for usage analytics.
- AI queries are stateless and not stored as part of user profiles.
